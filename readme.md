Css [![Build Status](https://travis-ci.org/devWayne/Css.svg?branch=master)](https://travis-ci.org/devWayne/Css)
============
> Css Handler 

##Usage

```
//setter
Css(Element,'width','200px');
Css(Element,{width:'200px',height:'200px'})

//getter
var width=Css(Element,'width');
```

##API

####CSS(el,obj)

####CSS(el,prop[,value])
